It's a project started for 'mgear' personal study. Many parts were referred to in 'mgear'(e.g api.core and UX/UI).
'mgear' is a great open source maya rigging framework.(https://github.com/mgear-dev/mgear4)

- Requirement  
    maya 2023.3
- maya `.mod`
